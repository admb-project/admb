Model description

The collection of examples following WinBUGS (Volume 1, WinBUGS 1.4) contains an application of Cox's proportional hazard model with random effects. A Weibull baseline hazard is used:

h0(t) = r*tr-1, r>0.

The "kidney data" contain information on time-to-infection for 38 patients, and there are several explanatory variables (see table below). A full description of the model can be found here: kidney.pdf.
