epil2 {glmmADMB}R Documentation

Seizure Counts for Epileptics

Description

Extended version of the 'epil' dataset of the MASS library. The three transformed variables Visit, Base and Age used by Booth et al. (2003) have been added to 'epil'.

Usage

data(epil2)

Format

A data frame with 236 observations on the following 12 variables.

y
a numeric vector
trt
a factor with levels placebo progabide
base
a numeric vector
age
a numeric vector
V4
a numeric vector
subject
a numeric vector
period
a numeric vector
lbase
a numeric vector
lage
a numeric vector
Visit
(rep(1:4,59)-2.5)/5
Base
log(base/4)
Age
log(age)

References

Booth et al. (2003) Negative binomial loglinear mixed models STATISTICAL MODELLING 3 (3): 179-191

Examples

data(epil2)
