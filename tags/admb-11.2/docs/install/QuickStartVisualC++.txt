#ADMB Project

*ADMB-11.2*  
*Released December 22, 2014*  

ADMB binary installation procedure for Windows and Visual C++ compilers using installer.

---
_**Note** &mdash; For Microsoft Visual C++ 2010, read [installation][help] documentation._
[help]: http://www.admb-project.org/documentation/installation/admb-installation-visual-c

Prerequisites
-------------

The following programs should be installed on the local computer.

* Visual Studio C++ installed and updated

    _**Note**_ &mdash; A free version is available from: [Visual Studio Express](http://www.visualstudio.com/downloads/download-visual-studio-vs#d-express-windows-desktop)

Quick Start
-----------

1. **Download** ADMB Windows Installer for Visual C++

       * For Visual C++ 2013 (64Bit), download [admb-11.2-windows8-vc12-64bit](http://ftp.admb-project.org/admb-11.2/admb-11.2-windows8-vc12-64bit.exe).

       * For Visual C++ 2013 (32Bit), download [admb-11.2-windows8-vc12-32bit](http://ftp.admb-project.org/admb-11.2/admb-11.2-windows8-vc12-32bit.exe).

       * For Visual C++ 2012 (64Bit), download [admb-11.2-windows8-vc11-64bit](http://ftp.admb-project.org/admb-11.2/admb-11.2-windows8-vc11-64bit.exe).

       * For Visual C++ 2012 (32Bit), download [admb-11.2-windows8-vc11-32bit](http://ftp.admb-project.org/admb-11.2/admb-11.2-windows8-vc11-32bit.exe).

2. **Install** ADMB

       Double click the downloaded **ADMB Windows Installer for Visual C++** and follow instructions.

3. **Use** ADMB
       
       On the Desktop, double click **ADMB Command Prompt** shortcut to open command line window.

       In the **ADMB Command Prompt** window, use the steps below to build and run the simple example.
 
       Copy the simple example directory

           C:\> xcopy /I /E "%ADMB_HOME%\examples\admb\simple" simple

       Change directory       

           C:\> cd simple

       Build simple example

           C:\simple\> admb simple

       Run simple example

           C:\simple\> simple

       Read manuals for more documentation.
