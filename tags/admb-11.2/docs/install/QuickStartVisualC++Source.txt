#ADMB Project

*ADMB-11.2*  
*Released December 22, 2014*  

ADMB source build and installation procedures for Windows and Visual C++ compilers.

---
_**Note** &mdash; For Microsoft Visual C++ 2010, read [installation][help] documentation._
[help]: http://www.admb-project.org/documentation/installation/admb-installation-visual-c

Prerequisites
-------------

The following programs are required to be installed on the local computer.

* Visual C++

    _**Note**_ &mdash; A free version is available from: [Visual Studio Express 2013](http://www.visualstudio.com/downloads/download-visual-studio-vs#d-express-windows-desktop)

Quick Start
-----------

1. **Download** ADMB source

       Download source distribution [admb-11.2-src.zip](http://ftp.admb-project.org/admb-11.2/admb-11.2-src.zip).

       Double click downloaded zip file and extract to **C:\admb\**.

2. Open **Visual Studio Command Prompt** window

       * For Visual C++ 2013, **Start**->**All Programs**->**Visual Studio**->**Visual Studio Tools**

       * For Visual C++ 2012, **Start**->**All Programs**->**Visual Studio**->**Visual Studio Tools**

3. **Build** ADMB source

       In the **Visual Studio Command Prompt** window, use the steps below to build ADMB.

       Change to admb directory

	    C:\> cd admb

       To build ADMB, use the commands below.

	    C:\admb\> nmake

       If debug symbols are needed, use the command below.

	    C:\admb\> nmake debug

       Close **Visual Studio Command Prompt**.

4. **Use** ADMB

       Using Windows Explorer, double click the **ADMB Command Prompt** shortcut located in **C:\admb\**.

       In the **ADMB Command Prompt** window, use the steps below to build and run the simple example.
 
       Copy the simple example directory

           C:\> xcopy /I /E "%ADMB_HOME%\examples\admb\simple" simple

       Change directory       

           C:\> cd simple

       Build simple example

           C:\simple\> admb simple

       Run simple example

           C:\simple\> simple

       Read manuals for more documentation.
