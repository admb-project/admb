#ADMB Project
*ADMB-11.2*  
*Released December 22, 2014*  

ADMB installation procedure for Windows and MinGW C++ compiler using zip file.

---

Prerequisites
-------------

The following programs should be installed on the local computer.

* MinGW C++ compiler installed

Quick Start
-----------

1. **Download**

       For Windows (64Bit), download [admb-11.2-windows8-mingw64](http://ftp.admb-project.org/admb-11.2/admb-11.2-windows8-mingw64.zip).
       For Windows (32Bit), download [admb-11.2-windows8-mingw32](http://ftp.admb-project.org/admb-11.2/admb-11.2-windows8-mingw32.zip).

2. **Extract**

       Double click downloaded file to open, then copy **admb** folder to **C:\>**.

5. **Use** ADMB

       Using Windows Explorer, double click the **ADMB Command Prompt** shortcut located in **C:\admb\**.

       In the **ADMB Command Prompt** window, use the steps below to build and run the simple example.
 
       Copy the simple example directory

           C:\> xcopy /I /E "%ADMB_HOME%\examples\admb\simple" simple

       Change directory       

           C:\> cd simple

       Build simple example

           C:\simple\> admb simple

       Run simple example

           C:\simple\> simple

       Read manuals for more documentation.
