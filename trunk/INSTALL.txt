Describe ADMB installation from source and binary distributions.

It is recommended to build ADMB from source to best match compiler and operating
system.

Requirements
============
ADMB has been tested and used on Windows and Linux operating systems in both 32
and 64 bit versions. A C/C++ compiler must be be installed on the local computer.

Below is a list of supported compilers:
 * GNU gcc version 4.7
 * Microsoft Visual C++ 2010 and 2012
 * MacOS XCode(with the Command Line Tools) version 4.6
 * Clang version 3.0
 * Intel C++ Compiler 11
 * Solaris Studio 12

If the compiler is not listed, try building from the source distribution. Email
users@admb-project.org for help.

Download
========
Source and pre-built binary distributions can be downloaded from
'http://www.admb-project.org/downloads/'.

Quick Start
===========

Windows
-------

1. Download ADMB source distribution.

   http://www.admb-project.org/buildbot/snapshots/admb-latest-src.zip

   Right click the downloaded file and extract to 'C:\admb-trunk\'.

2. Open Comand Prompt window and change to admb directory.

   C:\> cd admb-trunk

3. Install MinGW C++ compiler (32 Bit).

   C:\admb-trunk\> utilities\get-mingw.bat

   Download is approximately ~177M from 'sourceforge.net'.  This will also 
   update existing MinGW installation.

   Add MinGW to PATH.

   C:\admb-trunk\> set PATH=%CD%\utilities\mingw\bin;%PATH%

4. Build libraries

   C:\admb-trunk\> utilities\make

   Takes approximately 15-25 minutes to complete.

5. Build and run the simple example.

   C:\admb-trunk\> cd examples\admb\simple
   C:\admb-trunk\examples\admb\simple\> C:\admb-trunk\admb simple
   C:\admb-trunk\examples\admb\simple\> simple

Windows and Visual C++
----------------------

1. Download ADMB source distribution.

   http://www.admb-project.org/buildbot/snapshots/admb-latest-src.zip

   Right click the downloaded file and extract to 'C:\admb-trunk\'.

2. Open Visual Studio Comand Prompt window and change to admb directory.

   C:\> cd admb-trunk

3. Build libraries

   C:\admb-trunk\> nmake

   Takes approximately 15-25 minutes to complete.

4. Build and run the simple example.

   C:\admb-trunk\> cd examples\admb\simple
   C:\admb-trunk\examples\admb\simple\> C:\admb-trunk\admb simple
   C:\admb-trunk\examples\admb\simple\> simple

Unix
----

1. Download ADMB source distribution.

   http://www.admb-project.org/buildbot/snapshots/admb-latest-src.zip

2. Open Terminal windows.

3. Extract downloaded file in HOME directory.

   [~]$ unzip admb-latest-src.zip

3. Change to admb directory.

   [~]$ cd admb-trunk

4. Build libraries 

   [~/admb-trunk/]$ make

   Takes approximately 15-25 minutes to complete.

   Use help to show available compiler targets.

   [~/admb-trunk/]$ make help

6. Build run the simple example.

   [~/admb-trunk/]$ cd examples/admb/simple
   [~/admb-trunk/examples/admb/simple/]$ ~/admb-trunk/admb simple
   [~/admb-trunk/examples/admb/simple/]$ ./simple

7. [Optional] Installation

   'make install' is no longer supported.  Users should manually copy distribution
   to the preferred folder.

   Copy entire distribution.

   [~/admb-trunk/]$ sudo cp -Rvf ../admb-trunk /usr/local/admb 

   Create symlink to main admb script.  Directory '/usr/local/bin/' should exist.

   [~/admb-trunk/]$ sudo ln -sf /usr/local/admb/admb /usr/local/bin/admb

   After copying, build and run the simple example.

   [~/admb-trunk/]$ cd examples/admb/simple
   [~/admb-trunk/examples/admb/simple/]$ admb simple
   [~/admb-trunk/examples/admb/simple/]$ ./simple

Unix
----

1. Download ADMB binary distribution.

   NOT YET AVAILABLE. 

2. Open Terminal windows and change prefered installation directory.
  
   [~]$ cd /usr/local

3. Extract downloaded file.

   [/usr/local/]$ sudo unzip admb-latest.zip

   This will create admb directory '/usr/local/admb/'.

   Create symlink to main admb script.  Directory '/usr/local/bin/' should exist.

   [~/admb-trunk/]$ sudo ln -sf /usr/local/admb/admb /usr/local/bin/admb

3. Create shortcut softlink.

   [/usr/local/]$ sudo ln -sf admb/admb bin/admb

4. Build and run simple example.

   [~]$ cd
   [~]$ cp -R /usr/local/admb/example/admb/simple .
   [~/simple/]$ admb simple
   [~/simple/]$ ./simple
