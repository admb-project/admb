ADMB installation procedures for source and binary distributions.

Requirements
============
ADMB has been tested and used on Windows and Linux operating systems in both 32
and 64 bit versions. A C/C++ compiler must be be installed on the local computer.

Below is a list of supported compilers:
 * GNU gcc version 4.7
 * Microsoft Visual C++ 2012
 * Microsoft Visual C++ 2010
   - Read installation help 'http://www.admb-project.org/documentation/installation/admb-installation-visual-c'.
 * MacOS XCode(with the Command Line Tools) version 4.6
 * Clang version 3.0
 * Intel C++ Compiler 11
 * Solaris Studio 12

If the compiler is not listed, try building from the source distribution. For help,
email 'users@admb-project.org'.

Downloads
=========
Source and pre-built binary distributions can be downloaded from
'http://www.admb-project.org/downloads/'.

Quick Start
===========
* MinGW and Cygwin shells should use Unix installation procedure.
* Visual C++ users should use 'Windows and Visual C++' procedure.

Windows
-------

1. Download ADMB source distribution from

   'http://www.admb-project.org/downloads/'.

   Right click the downloaded file and extract to 'C:\admb\'.

2. Open 'Command Prompt' window and change to admb directory.

   C:\> cd admb

3. Install MinGW C++ compiler.

   If MinGW has been already installed, use the command below to add MinGW 
   programs to the system PATH.

   C:\> set PATH=C:\MinGW\bin;%PATH%

   If MinGW has not been installed, use the command below to download and 
   install 32-bit C++ compiler.

   C:\admb\> utilities\get-mingw.bat

   Downloads approximately 177 MB from 'sourceforge.net'.  This will also 
   update existing MinGW installation.

4. Build libraries

   C:\admb\> utilities\make

   To build with debug symbols,

   C:\admb\> utilities\make debug

   Takes approximately 15-25 minutes to complete.

5. Build and run the simple example.

   C:\admb\> cd examples\admb\simple
   C:\admb\examples\admb\simple\> C:\admb\admb simple
   C:\admb\examples\admb\simple\> simple

6. [Optional] Include admb in system 'PATH'.
   Avoids the need the to add the prefix path to the admb script. 

   A. In the 'System Properties' window, click on the 'Enviroment Variables' button.

   B. In the 'Enviroment Variables' windows, click on the 'New...' button, 
   in the 'Variable name:' type 

   PATH

   and in the 'Variable value:' type 

   C:\admb\;%PATH% 

   C. Click the 'OK' button when done and open a new command prompt.

   C:\> cd \admb\examples\admb\simple
   C:\admb\examples\admb\simple\> admb simple
   C:\admb\examples\admb\simple\> simple

Unix
----

1. Download ADMB source distribution from

   'http://www.admb-project.org/downloads/'.

2. Open 'Terminal' windows.

3. Extract downloaded file in HOME directory.

   [~]$ unzip admb-latest-src.zip

4. Change to admb directory.

   [~]$ cd admb

5. Build libraries 

   [~/admb/]$ make

   To build with debug symbols,

   [~/admb/]$ make debug

   To build shared library,

   [~/admb/]$ make shared

   Takes approximately 15-25 minutes to complete.

   Use help to show available compiler targets.

   [~/admb/]$ make help

6. Build run the simple example.

   [~/admb/]$ cd examples/admb/simple
   [~/admb/examples/admb/simple/]$ ~/admb/admb simple
   [~/admb/examples/admb/simple/]$ ./simple

7. [Optional] Installation

   'make install' is no longer supported.  Users should manually copy distribution
   to the preferred folder.

   Copy entire distribution.

   [~/admb/]$ sudo cp -Rvf ../admb /usr/local/admb 

   Create symlink to main admb script.  Directory '/usr/local/bin/' should 
   already exist.

   [~/admb/]$ sudo ln -sf /usr/local/admb/admb /usr/local/bin/admb

   After copying, build and run the simple example.

   [~/admb/]$ cd examples/admb/simple
   [~/admb/examples/admb/simple/]$ admb simple
   [~/admb/examples/admb/simple/]$ ./simple
